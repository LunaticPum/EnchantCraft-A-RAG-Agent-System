# DocumentPage 附魔书+书架重设计

## 设计目标
将 DocumentPage 从两栏面板改为 MC 附魔台风格——左侧雕纹书架 + 右侧附魔书。

## 组件清单

### 1. 雕纹书架 (Chiseled Bookshelf Grid)
- 3列CSS Grid等宽方块
- 每个方块 = 1个资料库(文件夹根目录)
- 方块内3×2书本槽位,文档数映射填充(0→空, 1-2→少量, 3-5→中等, 6+→满)
- 悬停金色虚线框 + tooltip(资料库名·文档数)
- 底部标签显示资料库名 + 文档计数

### 2. 附魔书 (Grimoire)
- 皮质封面(gradient #4a2a10→#3a1a08), 多层像素边框
- 四角金属包角(border + ◆铆钉)
- 骨白羊皮纸内页(repeating-linear-gradient + gradient)
- 左侧文档列表栏(160px) + 右侧内容预览
- 拖拽文档到书架方块触发迁移确认

### 3. 星空背景
- 来源: jibzdev/star-script (MIT)
- 双层 box-shadow 星点, 视差滚动(120s/200s)

### 4. 分页标签
- Pixelarticons 图标: Book Open / List Box / Message
- MC 像素按钮风格
- 三个分页共享书架背景

## 交互逻辑

### 资料库
- 散文件上传 → 归入"其他"资料库
- 目录上传 → 以根目录名自动创建资料库
- 点击+新建资料库
- 点击书架方块 → 右侧附魔书加载文档列表

### 删除
- 操作栏"删除" → 多选资料库 → 确认弹窗 → 级联删除文档+向量

### 迁移
- 附魔书侧栏文档可拖拽 → 拖到其他书架方块 → "将xxx迁移至xxx?" → 确认

## 技术实现

- CSS: 复用现有 MC 组件类(.mc-scene/.mc-btn 等)
- 字体: Monocraft (已加载)
- 图标: Pixelarticons (已安装)
- 星空: box-shadow 纯CSS(无需额外依赖)
- 木纹: SVG feTurbulence data-URL
- 拖拽: HTML5 Drag & Drop API

## 数据映射

| 字段 | 来源 |
|---|---|
| 资料库名 | source_document.directory_path 的根目录 |
| 文档数 | COUNT(source_document WHERE directory_path LIKE 'root%') |
| 分块数 | COUNT(document_chunk WHERE document_id IN ...) |
| 书本槽位填充 | 文档数 → 0~6映射 |
